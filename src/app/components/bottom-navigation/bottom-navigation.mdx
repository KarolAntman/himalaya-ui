# Bottom Navigation

Fixed `BottomNavigation` UI component designed for mobile devices, providing seamless and accessible navigation.

<Playground
  scope={{ BottomNavigation, Home, BarChart, Bell, Bookmark }}
  title={'Bottom Navigation '}
  desc={'A fixed footer navigation component for mobile devices'}

  <BottomNavigation transcluent={false}>
    <BottomNavigation.Item icon={ <Home />} url="#" />
    <BottomNavigation.Item icon={ <BarChart />} url="#"/>
    <BottomNavigation.Item icon={ <Bell />} url="#"/>
    <BottomNavigation.Item icon={ <Bookmark />} url="#"/>
</BottomNavigation>
`}
/>

<Attributes edit="/app/components/bottom-navigation.mdx">
<Attributes.Title>BottomNavigation.Props</Attributes.Title>

| Attribute   | Description                                         | Type      | Accepted values | Default |
| ----------- | --------------------------------------------------- | --------- | --------------- | ------- |
| transcluent | is header is using transcluent effect (transparent) | `boolean` | -               | `true`  |
| mobileOnly  | show only for mobile devices                        | `boolean` | -               | `true`  |

<Attributes.Title>BottomNavigation.Item.Props</Attributes.Title>

| Attribute  | Description                                 | Type             | Accepted values          | Default |
| ---------- | ------------------------------------------- | ---------------- | ------------------------ | ------- |
| icon       | icon to display                             | `icon`           | -                        | -       |
| url        | url for onclick                             | `string`         | -                        | `#`     |
| exactMatch | URL must exactly match the current pathname | `boolean`        | -                        | `true`  |
| ...        | native props                                | `HTMLAttributes` | `'id', 'className', ...` | -       |

</Attributes>
