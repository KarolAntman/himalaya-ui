# Show More

Styling component to show expanded or collapsed content.

<Playground
  title="General"
  desc="Basic Usage"
  scope={{ ShowMore, useState , Skeleton}}
  code={`
() => {
  const [expanded, setExpanded] = useState(false);
  return (
    <ShowMore expanded={expanded} onClick={() => setExpanded(!expanded)}>
      <Skeleton w={10} h={20} />
    </ShowMore>
  );
}
`}
/>

<Playground
  title="Custom"
  desc="Personalized title and Show/Hide lines."
  scope={{ ShowMore, useState, Skeleton }}
  code={`
() => {
  const [expanded, setExpanded] = useState(false);
  return (
    <div>
      <ShowMore 
        expanded={expanded} 
        onClick={() => setExpanded(!expanded)} 
        showLines={false} 
        showMoreTitle="Expand" 
        showLessTitle="Collapse"
      >
        <Skeleton w={10} h={20} mt={1}/>
      </ShowMore>
    </div>
  );
}
`}
/>


<Attributes edit="/app/components/show-more.mdx">
  <Attributes.Title>ShowMore.Props</Attributes.Title>

| Attribute       | Description                                          | Type              | Accepted Values                        | Default          |
|-----------------|------------------------------------------------------|-------------------|----------------------------------------|------------------|
| **expanded**    | Indicates whether content is expanded or collapsed.  | `boolean`         | `true` (expanded), `false` (collapsed) | `false`          |
| **onClick**     | Callback function to handle the click event.         | `() => void`      | -                                      | -                |
| **showLines**   | Controls the visibility of lines around the button.  | `boolean`         | `true`, `false`                        | `true`           |
| **showMoreTitle** | Title for the button when content is collapsed.   | `string`          | Any string                             | `"Show more"`    |
| **showLessTitle** | Title for the button when content is expanded.    | `string`          | Any string                             | `"Show less"`    |
| ...              | native props         | `HTMLAttributes`          | `'id', 'className', ...` | -       |

</Attributes>
