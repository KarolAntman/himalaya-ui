# Menu
Easily create and customize menus for your applications with the `Menu` UI component.

<Playground
  scope={{ Menu, Link }}
  title={'Menu example'}
  desc={'Example of a menu'}
  code={`
  () => {
     return (
    <Menu trigger="Menu">
			<Menu.Item>
				<Link href="/">Link Item 1</Link>
			</Menu.Item>
		</Menu>
    );
  };
  `}
/>

<Playground
  scope={{ Menu, Link, MoreHorizontal }}
  title={'Menu with custom trigger icon'}
  code={`
  () => {
     return (
    <Menu trigger={<MoreHorizontal />}>
			<Menu.Item>
				<Link href="/">Link Item 1</Link>
			</Menu.Item>
		</Menu>
    );
  };
  `}
/>

<Attributes edit="/app/components/menu.mdx">
<Attributes.Title>Menu.Props</Attributes.Title>

| Attribute  | Description              | Type                      | Accepted values          | Default       |
| ---------- | ------------------------ | ------------------------- | ------------------------ | ------------- |
| placement? | Placement of menu        | `Placement`               | -                        | `startBottom` |
| tigger?:   | Trigger element for menu | `ReactElement` / `string` | -                        | `Menu`        |
| ...        | native props             | `HTMLAttributes`          | `'id', 'className', ...` | -             |

<Attributes.Title>Menu.Item.Props</Attributes.Title>

| Attribute | Description  | Type             | Accepted values          | Default |
| --------- | ------------ | ---------------- | ------------------------ | ------- |
| ...       | native props | `HTMLAttributes` | `'id', 'className', ...` | -       |
