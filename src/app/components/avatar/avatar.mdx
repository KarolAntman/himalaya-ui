# Avatar

Avatars represent a user or a team. Stacked avatars represent a group of people.

<Playground

      <Avatar src={url} />

      <Avatar src={url} />
      <Spacer h={.5} />
      <Avatar src={url} isSquare />
      <Avatar src={url} isSquare />
      <Avatar src={url} isSquare />
      <Avatar src={url} isSquare />
    </>
  )
}
`}
/>
