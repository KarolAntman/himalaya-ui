# Gradient Word

The `GradientWord UI` component adds captivating gradient effects to text, enhancing your web application with dynamic and colorful typography.

<Playground
  scope={{ GradientWord }}
  title={'Gradient Word example'}
  desc={'Example of a gradient Word'}

  <GradientWord gradient={{degrees: '217deg', from: 'oklch(59.59% 0.24 255.09156059071347)', to: 'oklch(81.58% 0.189 190.74037768509325)'}} animationTime={'8000'}>
    Test
  </GradientWord>
  `}
/>

<Attributes.Title>GradientWord.Props</Attributes.Title>

| Attribute      | Description                        | Type             | Accepted values                    | Default                                                 |
| -------------- | ---------------------------------- | ---------------- | ---------------------------------- | ------------------------------------------------------- |
| gradient?      | degrees, starting and ending color | `object`         | `{ from: '', to: '', degrees: ''}` | `{ from: 'red', to: 'blue', degrees: 'to left bottom'}` |
| animationTime? | time to complete the animation     | `string`         | -                                  | `8000` (in milliseconds)                                |
| ...            | native props                       | `HTMLAttributes` | `'id', 'className', ...`           | -                                                       |

</Attributes>
