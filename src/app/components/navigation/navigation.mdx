# Header

**Navigation**

Effortlessly create navigation menus, including sub-menus and child items, using the `Navigation` UI component.

<Playground
  scope={{ Navigation }}
  title={'Navigation example'}
  desc={'Example of a navigation'}
  code={`
  <Navigation>
   <Navigation.Item title={'Item'}></Navigation.Item>
   <Navigation.Item title={'Item 2'}></Navigation.Item>
   <Navigation.Item title={'Item with childs'} >
      <Navigation.Item.Child title={'Home'}></Navigation.Item.Child>
      <Navigation.Item.Child title={'Home'}></Navigation.Item.Child>
   </Navigation.Item>
  </Navigation>
`}
/>

<Attributes.Title>Navigation.Props</Attributes.Title>

| Attribute         | Description                            | Type             | Accepted values          | Default |
| ----------------- | -------------------------------------- | ---------------- | ------------------------ | ------- |

| ...               | Native props                           | `HTMLAttributes` | `'id', 'className', ...` | -       |

<Attributes.Title>Navigation.Item.Props</Attributes.Title>

| Attribute   | Description                     | Type              | Accepted values          | Default |
| ----------- | ------------------------------- | ----------------- | ------------------------ | ------- |
| title       | Title of the navigation element | `string`          | -                        | -       |
| url         | Link for the navigation element | `string`          | -                        | -       |
| icon        | Icon for this menu item         | `React.ReactNode` | -                        | -       |
| exactMatch  | Matching url exactly            | `boolean`         | -                        | `true`  |
| columns     | number of columns for sub menu  | `boolean`         | -                        | `2`     |
| transcluent | transcluent effect              | `boolean`         | -                        | `true`  |
| ...         | native props                    | `HTMLAttributes`  | `'id', 'className', ...` | -       |
