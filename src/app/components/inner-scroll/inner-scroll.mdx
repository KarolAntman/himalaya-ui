# Scroll component

The `InnerScroll` component is a flexible element that empowers your content with scrollable capabilities.
Whether you need vertical, horizontal, or both directions, InnerScroll can adapt to your content requirements, making it an invaluable addition to your layout.

## Vertical scroll

<Playground
  title="Vertical scroll example by maxHeight"
  desc="Using maximum height"
  type={'vertical'}
  scope={{ InnerScroll, Skeleton }}
  code={`
() => {
  return (
    <InnerScroll style={{ maxHeight:150 }}  padding={1}>
     <Skeleton  height={10}></Skeleton>
    </InnerScroll>
  )
}
`}
/>

<Playground
  title="Vertical scroll example by height"
  desc="Using fixed height"
  type={'vertical'}
  scope={{ InnerScroll, Skeleton }}
  code={`
() => {
  return (
    <InnerScroll height={5} padding={1}>
      <Skeleton height={10}></Skeleton>
    </InnerScroll>
  )
}
`}
/>
