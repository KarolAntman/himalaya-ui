<Attributes.Title>FadeInEffect.Props</Attributes.Title>

| Attribute     | Description                                                          | Type             | Accepted values          | Default |
| ------------- | -------------------------------------------------------------------- | ---------------- | ------------------------ | ------- |
| transition?   | Animation transition in milliseconds                                 | `number`         | -                        | `750`   |
| delay?        | Delay before starting the animation                                  | `number`         | -                        | `0`     |
| blur?         | Blur effect                                                          | `number`         | -                        | `0`     |
| startOpacity? | Starting opacity of element                                          | `number`         | -                        | `0`     |
| rotate?       | Rotate css property                                                  | `string`         | -                        | `0`     |
| translateX?   | Translate the component horizontally (CSS translateX transformation) | `string`         | -                        | `0`     |
| translateY?   | Translate the component vertically (CSS translateY transformation)   | `string`         | -                        | `0`     |
| skewX?        | Skew component horizontally (CSS skewX transformation)               | `string`         | -                        | `0`     |
| skewY?        | Skew component horizontally (CSS skewY transformation)               | `string`         | -                        | `0`     |
| scaleX?       | Scale component horizontally (CSS scaleX transformation)             | `string`         | -                        | `1`     |
| scaleY?       | Scale component vertically (CSS scaleY transformation)               | `string`         | -                        | `1`     |
| ...           | native props                                                         | `HTMLAttributes` | `'id', 'className', ...` | -       |

</Attributes>
