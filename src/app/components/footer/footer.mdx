# Footer

The `Footer` UI component is a versatile element designed to be placed at the bottom of a web page or section, typically used for navigation, contact information, or additional content.

<Attributes.Title>Footer.Block.Props & FooterBottom.Block.Props</Attributes.Title>

| Attribute | Description       | Type             | Accepted values                                                                        | Default      |
| --------- | ----------------- | ---------------- | -------------------------------------------------------------------------------------- | ------------ |
| justify?  | position of block | `string`         | `'flex-start', 'center', 'flex-end', 'space-between', 'space-around', 'space-evenly',` | `flex-start` |
| ...       | native props      | `HTMLAttributes` | `'id', 'className', ...`                                                               | -            |

<Attributes.Title>FooterNavigation.Props</Attributes.Title>

| Attribute | Description                  | Type             | Accepted values          | Default |
| --------- | ---------------------------- | ---------------- | ------------------------ | ------- |
| title     | Heading of Footer Navigation | `string`         | -                        | -       |
| ...       | native props                 | `HTMLAttributes` | `'id', 'className', ...` | -       |

<Attributes.Title>FooterNavigation.Item.Props</Attributes.Title>

| Attribute | Description                     | Type             | Accepted values                | Default |
| --------- | ------------------------------- | ---------------- | ------------------------------ | ------- |
| href      | Link for navigation item        | `string`         | -                              | -       |
| target    | Target property for anchor link | `string`         | `_blank, _self, _parent, _top` | `_self` |
| ...       | native props                    | `HTMLAttributes` | `'id', 'className', ...`       | -       |

</Attributes>
