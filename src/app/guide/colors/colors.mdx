# Colors

Default colors for themes.

## How to

You can use any color anywhere, all colors change with the theme.

<Display w="90%" caption={<span>Get var from global css.</span>}>

```jsx
const myComponent = () => {
  return <p style={{ color: `var(--color-background-800)` }}>myComponent</p>;
};
```

</Display>
